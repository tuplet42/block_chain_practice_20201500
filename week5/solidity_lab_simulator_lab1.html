Lab 1. HelloWorld & HelloNumber
Remix IDE 느낌의 3단 시뮬레이터 · public/private · getter/setter · int 실습
기초 ★☆☆ ~15분 string public view int

탭 1은 public 변수의 자동 getter를 확인하고, 탭 2는 private + 함수 조합을 체험합니다. 탭 3은 HelloNumber 기본, 탭 4는 HelloNumber 확장, 탭 5는 HelloWorld + HelloNumber 결합 실습입니다.

실습 포인트
  1. 코드 확인 → Solidity 문법 하이라이팅과 줄 번호를 봅니다.
  2. 컴파일 → pragma/contract/중괄호/기본 구조를 검사합니다.
  3. 배포 → 가상 VM에 컨트랙트를 올리고 함수 호출 UI를 생성합니다.
  4. 호출 → view는 무료, 상태 변경 함수는 gas를 차감합니다.
  5. 확장 실습 → 탭 4에서 HelloNumber 확장, 탭 5에서 결합 컨트랙트를 확인합니다.
